## Jellyfish Boids

### Game Description

Jellyfish Boids is a flocking simulation based on the Boids algorithm
taught in class. Jellyfish move through an underwater environment using
separation, alignment, and cohesion behaviors to simulate natural group
movement.

The player can influence the flock using an attractor while the
jellyfish navigate around obstacles in the environment.

------------------------------------------------------------------------

### How to Interact

Move the attractor to influence where the jellyfish flock moves.

Boids automatically:

-   Align with nearby jellyfish\
-   Avoid crowding each other\
-   Move toward the group center\
-   Avoid obstacles in the environment

------------------------------------------------------------------------

# Features

Base Boids flocking behavior (separation, alignment, cohesion)

Title screen before entering the simulation

Obstacle avoidance system

Attractor that influences flock movement




# Extra Credit Features

Underwater theme with jellyfish models and ocean environment

Boid motion trails for visual movement effects

Bubble particle effects 

Animated seaweed environment

Dynamic obstacle spawning during gameplay
